14. FLAGS DISPONIBILI NELLA CLI
text

--mode          ModalitÃ : full, train, predict, analyze, chat, interactive
--data          Percorso file dati CSV specifico
--data-dir      Directory dati (default: "data")
--model-type    Tipo modello: standard, stacked, bidirectional
--epochs        Numero epoche training
--batch-size    Dimensione batch
--api-key       API key DeepSeek
--config        File configurazione JSON
--input-data    File dati per predizione specifica
--generate-report Genera report AI automaticamente
